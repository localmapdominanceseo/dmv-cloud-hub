# DMV Content Style Rules

Do not build every article with the same pattern.

Use different guide types:
- Renewal guide
- Replacement guide
- REAL ID checklist
- Fee explainer
- Online eligibility guide
- Registration guide
- Mistakes to avoid
- Documents checklist

Workflow:
1. Add `.md` files to `content/guides/`
2. Run `python generate.py`
3. Commit and push with GitHub Desktop
4. Cloudflare deploys `/guides/slug/`
